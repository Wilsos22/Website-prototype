❓ How to Play

Solo Blast. Problem blocks fall and stack from the bottom. Type the answer to a block and press Enter — the lowest block with that answer explodes. Clear them faster than they pile up. If a column reaches the very top, it's game over.

Build-It. Blocks show single numbers and you get a TARGET. Tap blocks so they add up to the target, then they all blast. Great for number bonds and breaking numbers apart.

Combos. Answer correctly in a row to build a multiplier — more streak, more points and coins.

Power-ups. Earn them by keeping streaks:
❄️ Freeze (key Q) — stops the blocks for 5 seconds.
💥 Bomb (key W) — clears the whole bottom row.
💡 Hint (key E) — reveals an easy block to clear.

Earn & unlock. Coins unlock new color themes. Badges and ranks track your progress over time.

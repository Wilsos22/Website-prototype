One system, eleven meanings: identical warm-dotted ground, timer, and layout on every state — the only thing that changes is the accent, so the colour itself tells the room where they are (teal = warm-up/review, orange = launch, green = concrete, blue = representational, violet = abstract, amber = learning check, coral = discussion, brown = independent, rose = exit ticket, gold = closeout). Math-meaning colours (green prime, slate composite) stay constant underneath, independent of the state accent.
